File handling

File handling is a crucial aspect of programming, allowing you to read from and write to files on your computer. Python provides built-in functions and methods to perform file handling operations. Here's an overview of basic file handling in Python:

1.opening a file
syntax

file=open("filename","mode")

Example

#textfile
tx.txt
how are you?

f=open("tx.txt","r")

2.reading a file
syntax

file=open("filename","mode")

Example

#yu.txt
hi
how are you

f=open("yu.txt","r")
o=f.read()
print(o)
output:hi
how are you

3.reading the file upto given index
Syntax

#ui.txt
hello hi

Example

f=open("ui.txt","r")
o=f.read(5)
print(o)
output:hello

4.reading the file line by line

#yu.txt
hi
how are you

f=open("yu.txt","r")
o=f.readline()
print(o)
output:hi
how are you

4.reading the file in list

#yu.txt
hi
how are you

f=open("yu.txt","r")
o=f.readlines()
print(o)
output:["hi","how are you"]

5.writing on file
syntax

file=open("filename","mode")
file.write("hello")

Example

f=open("tx.txt","w")
f.write("hello")

6.closing a file
syntax

file.close()
