variables

In Python, a variable is a named storage location that holds a value. It acts as a symbolic name for a value, making it easier to reference and manipulate data in your program. Unlike some other programming languages, in Python, you don't need to explicitly declare the type of a variable; the interpreter infers the type based on the value assigned to it.

Here's a simple example:

# Variable assignment
x = 10
# Printing the value of the variable
print(x)

Python variable naming rules:

  • Variable names are case-sensitive (myVar and myvar are different variables).
  • Variable names can contain letters, numbers, and underscores.
  • Variable names cannot start with a number.
  • Python has some reserved words (keywords) that cannot be used as variable names because they have special meanings in the language.

# Valid variable names
my_variable = 42
user_input = "Hello, World!"
# Invalid variable names (due to starting with a number)
1st_variable = 5
