Function

In Python, a function is a block of reusable code that performs a specific task. Functions are defined using the def keyword, and they allow you to modularize your code by breaking it into smaller, more manageable pieces. Here's the basic syntax for defining a function:

def function_name(parameters):
# code inside the function
# perform some task
return result # optional, can return a value

Function name: A name that you choose to identify your function.
Parameters: Input values that the function takes. These are optional.
Function body: The block of code that performs the desired task.
Return statement: An optional statement used to return a value from the function.

Function without return type

Example:
def fun():
print("hello")
fun()
output:hello

Function with return type

Example:
def fun():
return "hello"
k=fun()
print(k)
output:hello

Function with passing arguments

Example:
def fun(a,b):
c=a+b
print(c)
fun(2,3)
output:5

Function without passing arguments

Example:
def fun():
a=3
b=3
print(a+b)
output:6
