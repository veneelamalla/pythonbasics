Iterative statements

Iterative statements, also known as loops, in Python are used to repeatedly execute a block of code. The two main types of loops in Python are for and while loops.

1. for loop:

The for loop is used to iterate over a sequence (such as a list, tuple, string, or range) and execute a block of code for each item in the sequence.

using for loop in list iteration

h=[1,2,3,,4] #list declaration
for i in h:
print(i)
output: 1 2 3 4

using for loop in string iteration

h="apple"
for i in h:
print(i)
output:a
p
p
l
e

using for loop using range function

#print 1 to 5 numbers using range function
for i in range(1,6):
print(i)
output: 1 2 3 4 5

2.while loop

The while loop is used to repeatedly execute a block of code as long as a specified condition is True.

#print 1 to 5 number
i=1
while(i<=5):
print(i)
i=i+1
output:1 2 3 4 5

break statement

It is used to exit the loop prematurely based on a certain condition.

for i in range(10):
if i == 5:
break
print(i)
output: 0 1 2 3 4

continue statement

It is used to skip the rest of the code inside the loop for the current iteration and move to the next iteration.

for i in range(5):
if i == 4:
continue
print(i)
output: 0 1 2 3 5
