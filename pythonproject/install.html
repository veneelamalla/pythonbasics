Installation and Setup

For Windows:

1.Visit the Python Official Website:

Go to the Python Downloads page.

2.Download Python:

Click on the "Download Python" button. Ensure you download the latest version.

3.Run the Installer:

Locate the downloaded installer and run it. Check the box that says "Add Python to PATH" during installation.

4.Installation:

Click "Install Now" and let the installer run.

5.Verification:

Open a command prompt and type python --version or python -V to verify the installation.

For Linux (Ubuntu/Debian):

1.Open Terminal:

Open a terminal using Ctrl+Alt+T.

2.Update Package List:

Run the following command to update the package list:

sudo apt update

3.Install Python:

Run the following command to install Python 3:

sudo dnf install python3

4.Verification:

Type python3 --version or python3 -V to verify the installation.
